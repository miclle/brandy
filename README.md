# brandy

[![Build Status](https://travis-ci.org/miclle/brandy.svg?branch=master)](https://travis-ci.org/miclle/brandy)

## Usage:
  brandy new APP_PATH

## Commands:

### brandy

### help, h

Display help text

### new, n

Create

### clean, c

### build, b

### server, s

## Global Options:

### -port, --port

Specify a port to use


## Example:
  brandy new ~/code/app
