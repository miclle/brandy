# brandy

## Usage:
  brandy new APP_PATH

## Commands:

### brandy

### help, h

Display help text

### new, n

Create

### clean, c

### build, b

### server, s

### verbose, v

Verbose mode. A lot more information output.

### version, V

Print the brandy version. Combine with verbose for more info.


## Global Options:

### -port, --port

Specify a port to use


## Example:
  brandy new ~/code/app
