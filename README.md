# lisa

[![Build Status](https://travis-ci.org/miclle/lisa.svg?branch=master)](https://travis-ci.org/miclle/lisa)

## Usage:
  lisa new APP_PATH

## Commands:

### lisa

### help, h

Display help text

### new, n

Create

### clean, c

### build, b

### server, s

## Global Options:

### -port, --port

Specify a port to use


## Example:
  lisa new ~/code/app
